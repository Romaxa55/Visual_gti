# GTI Data Visualizer

Интерактивный визуализатор данных ГТИ с возможностью симуляции и настройки параметров.

## Возможности

- 📊 **Трёхпалеточная визуализация** - отображение данных в трёх вертикальных палетках
- 📁 **Поддержка множества форматов** - Excel (xlsx, xls), CSV, LAS файлы
- 🎮 **Режим симуляции** - пошаговая прорисовка данных с настраиваемой скоростью
- ⚙️ **Настройка параметров** - изменение цветов, единиц измерения, диапазонов
- 🔍 **Интерактивная фильтрация** - фильтрация по глубине и времени
- 💻 **Кроссплатформенность** - работает на Windows, Mac, Linux

## Системные требования

- Node.js 18+ 
- npm или yarn
- Electron

## Установка и запуск

### Для разработки

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd gti-data-visualizer
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите в режиме разработки:
```bash
npm run dev
```

### Для сборки

1. Соберите приложение:
```bash
npm run build
```

2. Создайте дистрибутив:
```bash
# Для Windows
npm run dist:win

# Для Mac
npm run dist:mac

# Для Linux
npm run dist:linux
```

Готовые файлы будут в папке `release/`

## Поддерживаемые форматы данных

### Excel файлы (.xlsx, .xls)
- Первая строка должна содержать заголовки
- Данные начинаются со второй строки
- Рекомендуется иметь колонку с глубиной

### CSV файлы (.csv)
- Разделитель - запятая
- Первая строка - заголовки
- Кодировка UTF-8

### LAS файлы (.las)
- Стандартный формат каротажа
- Поддерживается секция ~ASCII
- Автоматическое определение параметров

## Использование

### Загрузка данных
1. Нажмите **"Загрузить файл"** в заголовке
2. Выберите файл данных
3. Данные автоматически обработаются и отобразятся

### Режим симуляции
1. Нажмите **"Старт"** для запуска симуляции
2. Настройте скорость с помощью слайдера
3. Используйте **"Пауза"** для остановки

### Настройка параметров
1. Нажмите **"Настройки параметров"**
2. Измените цвета, единицы измерения, диапазоны
3. Включите/выключите отображение параметров
4. Нажмите **"Сохранить"**

### Фильтрация данных
- Используйте слайдер **"Диапазон глубины"** для фильтрации
- Настройте текущий индекс для точного позиционирования

## Особенности

### Трёхпалеточная система
- Параметры автоматически распределяются по трём палеткам
- Каждая палетка может содержать несколько параметров
- Цвета и настройки индивидуальны для каждого параметра

### Автоматическое определение данных
- Глубина: ищется по ключевым словам (depth, dept, глубина)
- Время: ищется по ключевым словам (time, время, час)
- Параметры: все остальные численные колонки

### Режим симуляции
- Пошаговая прорисовка данных
- Настраиваемая скорость (100-3000мс)
- Индикатор текущей позиции
- Отображение текущих значений

## Горячие клавиши

- **Ctrl+O** (Cmd+O) - Открыть файл
- **Ctrl+S** (Cmd+S) - Переключить режим симуляции
- **Ctrl+P** (Cmd+P) - Открыть настройки параметров
- **Ctrl+Q** (Cmd+Q) - Выход

## Устранение неполадок

### Приложение не запускается
- Убедитесь, что установлена правильная версия Node.js
- Проверьте установку зависимостей: `npm install`

### Ошибки при загрузке файлов
- Проверьте формат файла
- Убедитесь, что файл содержит численные данные
- Проверьте кодировку (должна быть UTF-8)

### Проблемы с отображением
- Проверьте настройки параметров
- Убедитесь, что параметры видимы
- Проверьте диапазон глубины
